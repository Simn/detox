Widget Demo



Notify + Progress

This page demonstrates the use of "Widgets" in DOMTools. Basically, they allow you to create a haxe class that correlates to a DOM object. In this example, I have a notification widget that is created based on this form, and a progress bar widget, which is based on a random number.

Have a look at the code to see how it works. Also note you can interact with the widgets in a small way: the notifications you can close, the progress bars you can click to update the value with a new random number.
